🚀 راه‌اندازی (۵ دقیقه)

برای داشتن سرویس شخصی خودت، مراحل زیر را انجام بده:

۱. ربات تلگرام بساز
توی تلگرام برو به @BotFather و دستور /newbot را بزن. اسم ربات را مثلاً ShorayeKheradBot بگذار. توکنی که می‌گیری را ذخیره کن.
۲. کلید API میسترال بگیر
برو به console.mistral.ai، ثبت‌نام کن و یک کلید API جدید بساز. کلید با sk-mi شروع می‌شود. رایگان: Mistral برای کاربران جدید ۱ میلیون توکن رایگان می‌دهد.
۳. ورکر را روی کلودفلر دیپلوی کن
کل پروژه را از گیت‌هاب کلون کن و مراحل زیر را در ترمینال اجرا کن: 
# Clone
git clone https://github.com/YOUR_USER/council-hub.git
cd council-hub/worker

# Install dependencies
npm install

# Create KV namespaces and copy IDs
npx wrangler kv:namespace create KEYS
npx wrangler kv:namespace create SESSIONS

# Edit wrangler.toml — paste the KV IDs
# Set secrets
npx wrangler secret put TELEGRAM_BOT_TOKEN
# Paste your bot token when prompted

npx wrangler secret put COUNCIL_SECRET
# Generate: openssl rand -hex 32

# Deploy
npx wrangler deploy
۴. وب‌هوک تلگرام را تنظیم کن
بعد از دیپلوی، آدرس ورکرت را از خروجی wrangler deploy بردار (مثلاً council-worker.YOUR.workers.dev) و اجرا کن: 
curl "https://api.telegram.org/bot<TOKEN>/setWebhook?url=https://council-worker.YOUR.workers.dev/"
۵. به ربات خودت پیام بده
توی تلگرام رباتت را باز کن و /start بزن. بعد کلید API را تنظیم کن: 
/setkey sk-mi-YOUR_MISTRAL_KEY
حالا می‌توانی سوال بپرسی: 
/council --full آینده هوش مصنوعی در ایران

فازهای جلسه

🔵 افتتاحیه (فارابی)
تحلیل مسئله با چهار علت: مادی، صوری، فاعلی، غایی. معرفی اعضا.
💬 نظرات اولیه
هر حکیم به‌نوبت نظر خود را با ارجاع به آثارش بیان می‌کند.
⚡ گفتگو و نقد (۲ دور)
اعضا به صورت تصادفی یکدیگر را نقد می‌کنند. مناظره زنده.
📋 صورت‌جلسه (بیرونی)
دبیر جلسه نقاط اشتراک و اختلاف را ثبت می‌کند.
🏆 رأی نهایی (فارابی)
رئیس جمع‌بندی می‌کند: اجماع / اکثریت / فاقد اجماع.

معماری فنی

ورکر کلودفلر وب‌هوک تلگرام را دریافت می‌کند. هر جلسه ۲-۳ بار مدل mistral-large-latest را با پرامپت‌های دسته‌ای صدا می‌زند (نه ۲۵ بار). خروجی هر حکیم بلافاصله به تلگرام فرستاده می‌شود. کل جلسه در کمتر از ۱۵ ثانیه کامل می‌شود.

💰 هزینه

Cloudflare Workers: رایگان (Free Plan ۱۰۰k req/day)
Cloudflare KV: رایگان (تا ۱GB)
Mistral AI: هزینه هر جلسه ~۰.۰۰۲ دلار (۲۵۰ جلسه = ~۰.۵ دلار). کاربران جدید ۱M توکن رایگان دارند.
جمع: ماهی ~۰.۵ دلار برای استفاده معمولی.
